# Security Policy

## Supported Versions

Currently, the following versions of EmDash are supported with security updates:
| Version | Supported          |
| ------- | ------------------ |
| v0.x.x  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in EmDash, please do **NOT** open a public issue.
Instead, send an email to security@emdashcms.com or use GitHub's private vulnerability reporting feature for this repository.

All security vulnerabilities will be promptly addressed.
